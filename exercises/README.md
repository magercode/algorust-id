# Exercise Pack ALGORUST

Gunakan urutan berikut agar progres terstruktur:

1. `ex01_sorting.rs` - implementasi merge sort.
2. `ex02_graph_bfs.rs` - hitung jarak hop dengan BFS.
3. `ex03_knapsack.rs` - 0/1 knapsack dynamic programming.
4. `ex04_gossip.rs` - simulasi sebar pesan pada jaringan P2P.
5. `ex05_crypto.rs` - Caesar decrypt, XOR roundtrip, dan mini Diffie-Hellman.

## Cara Pengerjaan

1. Salin file dari `exercises/starter/` ke area kerja Anda.
2. Implementasikan bagian `TODO`.
3. Bandingkan pendekatan dengan referensi di `exercises/solutions/`.
4. Jalankan `cargo test` dan pastikan tidak ada regresi.

## Rubrik Penilaian (Skala 1-5)

- Correctness: hasil sesuai expected output.
- Complexity: pemilihan algoritma sesuai konteks.
- Rust quality: idiomatik, aman, dan mudah dibaca.
- Testing mindset: mampu menguji edge case penting.
